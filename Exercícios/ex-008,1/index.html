Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Nessa frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica)

Nessa frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar um trecho marcado como se fosse um marca texto com a tag <mark>

Esse outro trecho é para teste de style (CSS) para mudar a cor do marcador.

Texto grande e pequeno

Podemos aumentar e diminuir o texto usando as tags <big> (tag obsoleta) e <small>

Texto deletado

Para criar um texto deletado use a tag <del>

Texto inserido

Para um texto inserido ou sublinhado use a tag <ins>

Texto sobrescito

Para inserir algo sobrescrito como uma potência (x20+2) use a tag <sup>

Texto subscrito

Para criar um texto subscrito como uma equação química (H2O) use a tag <sub>

Texto pontilhado

Para textos pontilhados use a tag <abbr>
